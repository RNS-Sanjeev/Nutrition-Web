# Nutrition-Web
NutriBalance — responsive nutrition &amp; dietitian booking UI (Login, Search, Profiles, Booking) — a modern React/Tailwind frontend.
NutriBalance is a responsive frontend for finding and booking certified nutritionists and dietitians. It includes a polished login/signup flow, role selector, search and filters, practitioner cards with ratings/pricing, and diet-tracking widgets — built for a clean, mobile-first user experience.
